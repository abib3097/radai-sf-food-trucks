# Mobile Food Service data extracted from [Here](https://data.sfgov.org/Economy-and-Community/Mobile-Food-Facility-Permit/rqzj-sfat/data)
Place `Mobile_food_Facility_Permit.csv` in this directory and rename it to `food_trucks.csv` in order to match the file
name in`app/services/truck_service.py`. That is the file name the service expects.

Note: This is just done for naming simplicity, in production I would make this filename a constant and just keep it as is.
